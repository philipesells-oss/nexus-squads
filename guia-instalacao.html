Guia de Instalação
Nexus Squads

Instale 12 times de agentes IA especialistas no seu Claude Code. Este guia cobre tudo: do zero absoluto até os squads funcionando.

⏱️ Tempo estimado: ~15 minutos
💻 Sistemas: Mac · Windows · Linux
🎯 Nível: Do zero ao avançado
⚡ O que você vai precisar
  • ○ Node.js 18+ (instalamos no Passo 0a)
  • ○ Git (instalamos no Passo 0b)
  • ○ Claude Code CLI (instalamos no Passo 1)
  • ○ Conta na Anthropic com API key (Passo 2)
  • ○ Conexão com internet
  • ○ ~500MB de espaço em disco
ℹ️

Se você já tem Node.js, Git e Claude Code instalados, pode pular direto para o Passo 3.

0a
Pré-Requisito

Instalar o Node.js

O Node.js é necessário para instalar o Claude Code via npm. É gratuito e você instala em 2 minutos.

💡

Opção recomendada para Mac: Use o instalador oficial do site nodejs.org — é o mais simples. Ou use o Homebrew se já tiver instalado.

Opção A — Instalador Oficial (mais fácil):

1. Acesse nodejs.org e clique em "LTS Download"

2. Baixe o .pkg para macOS

3. Abra o arquivo e siga as instruções (próximo, próximo, instalar)

4. Reinicie o terminal após instalar

Opção B — Homebrew (terminal):

bash 
# Se não tiver Homebrew, instale primeiro:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Depois instale o Node.js:
brew install node

1. Acesse nodejs.org

2. Clique em "LTS Download" — baixa o instalador .msi

3. Execute o instalador e clique Next em todas as telas

4. Marque a opção "Automatically install necessary tools" quando aparecer

5. Reinicie o computador após instalar

⚠️

Windows: Use sempre o Command Prompt (CMD) ou PowerShell como Administrador para executar comandos.

bash — Ubuntu/Debian 
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
✅

Verificar instalação: Abra o terminal e rode node --version. Deve aparecer algo como v20.x.x.

Resultado Esperado

O comando node --version retorna um número de versão (ex: v20.11.0) e o npm --version também.

0b
Pré-Requisito

Instalar o Git

O Git é necessário para baixar os repositórios dos Nexus Squads. Gratuito e disponível para todos os sistemas.

💡

Mac: O Git geralmente já está instalado. Rode git --version no terminal. Se aparecer um número de versão, pule para o Passo 1.

bash 
# Via Homebrew (recomendado):
brew install git

# Ou rode este comando — instala automaticamente via Xcode tools:
git --version
# Se pedir para instalar, clique em "Install" na janela que abre

1. Acesse git-scm.com e clique em "Download for Windows"

2. Execute o instalador e clique Next mantendo as opções padrão

3. Na tela "Adjusting PATH", selecione "Git from the command line..."

4. Conclua a instalação e reinicie o terminal

bash — Ubuntu/Debian 
sudo apt-get update && sudo apt-get install -y git
Resultado Esperado

O comando git --version retorna algo como git version 2.43.0.

1
Passo 1

Instalar o Claude Code

O Claude Code é o terminal de IA da Anthropic. É aqui onde seus Nexus Squads vão viver. Instalação via npm em um único comando.

bash — Mac / Windows / Linux 
# Instala o Claude Code globalmente via npm
npm install -g @anthropic-ai/claude-code
⚠️

Windows: Se aparecer erro de permissão, abra o PowerShell como Administrador (botão direito no ícone → "Executar como administrador") e rode o comando novamente.

💡

Mac com erro de permissão? Rode: sudo npm install -g @anthropic-ai/claude-code e digite sua senha quando pedido.

Verificar a instalação:

bash 
claude --version
# Deve aparecer: @anthropic-ai/claude-code@x.x.x
Resultado Esperado

O comando claude --version retorna o número da versão instalada. Se aparecer isso, o Claude Code está instalado.

2
Passo 2

Configurar a API Key da Anthropic

Para usar o Claude Code você precisa de uma conta na Anthropic e uma API key. O processo é simples e gratuito para começar.

🔑 Criar sua conta e API Key
  • 1 Acesse console.anthropic.com e crie uma conta gratuita
  • 2 No painel, clique em "API Keys" no menu lateral
  • 3 Clique em "Create Key", dê um nome (ex: "claude-code")
  • 4 Copie a chave gerada — ela começa com sk-ant-...
  • 5 Guarde em local seguro — ela aparece apenas uma vez

Autenticar o Claude Code:

bash 
# Inicie o Claude Code — ele vai pedir sua API key automaticamente
claude

# Quando pedir: cole sua API key (sk-ant-...) e pressione Enter
🔒

Segurança: Nunca compartilhe sua API key. Ela dá acesso à sua conta e ao seu saldo. Se vazar, vá em console.anthropic.com e revogue imediatamente.

💳

Sobre o custo: A Anthropic oferece créditos gratuitos para começar. Para uso intenso, você pode adicionar um método de pagamento. O custo é por tokens usados — não por mensalidade.

Resultado Esperado

Ao rodar claude no terminal, o Claude Code abre e você pode conversar com ele. Se aparecer o prompt interativo, tudo certo.

3
Passo 3

Instalar o AIOS (Base)

O AIOS é o framework base que ativa os agentes dev, qa, pm, po, sm, architect e mais. É a fundação sobre qual os Nexus Squads operam.

📌

Como usar este passo: Abra o terminal. Copie e cole cada bloco de comandos em sequência, esperando o anterior terminar antes de rodar o próximo.

Clonar o repositório AIOS:

bash — Terminal 
git clone https://github.com/SynkraAI/aios-core.git ~/aios-core

Instalar as dependências:

bash — Terminal 
cd ~/aios-core && npm install && npm link

Copiar os comandos para o Claude Code:

bash — Terminal 
cp -r ~/aios-core/.claude/commands/AIOS ~/.claude/commands/AIOS
💡

O que acontece: Os agentes AIOS (dev, qa, pm, po, sm, architect e outros) ficam disponíveis globalmente em todos os projetos via slash commands no Claude Code.

Resultado Esperado

Nenhum erro no terminal. A pasta ~/.claude/commands/AIOS foi criada com os agentes dentro.

4
Passo 4

Instalar os Nexus Squads

Aqui é onde a mágica acontece. Dois comandos e você tem 12 times de especialistas em IA prontos para trabalhar no seu Claude Code.

Clonar o repositório dos Squads:

bash — Terminal 
git clone https://github.com/ohmyjahh/xquads-squads.git ~/nexus-squads

Copiar os squads para o Claude Code:

bash — Terminal 
cp -r ~/nexus-squads/squads/* ~/.claude/commands/
🚀

O que acontece: Os 12 squads ficam disponíveis: Advisory-Board, Brand-Squad, Copy-Master, Cybersecurity, Data-Squad, Design-Squad, Hormozi-Squad, Movement, Storytelling, Traffic-Masters, C-Level-Squad, Claude-Code-Mastery.

🔄

Para atualizar no futuro: Rode cd ~/nexus-squads && git pull e depois repita o comando cp para sobrescrever com a versão mais nova.

Resultado Esperado

Nenhum erro no terminal. Os 12 squads estão na pasta ~/.claude/commands/.

Passo 5

Verificar a Instalação

Hora de confirmar que tudo está funcionando. Feche o Claude Code, abra novamente e veja seus squads.

📋 Sequência de verificação
  • 1 Feche o Claude Code completamente (se estiver aberto)
  • 2 Abra qualquer pasta no terminal: cd ~/Desktop
  • 3 Execute claude para abrir o Claude Code
  • 4 Digite / e pressione Enter (ou Tab)
  • 5 Você deve ver a lista de todos os squads e agentes AIOS

Verificar via Claude Code (cole este prompt):

COLE NO CLAUDE CODE 
Liste todos os slash commands disponíveis para confirmar que o AIOS e os Nexus Squads estão instalados.
Resultado Esperado

Ao digitar / você verá os agentes AIOS (/AIOS/dev, /AIOS/qa...) + os 12 squads (/advisory-board, /brand-squad, /copy-master...) disponíveis em qualquer projeto.

🎉

Parabéns! Se você chegou até aqui e os squads aparecem, você completou a instalação. Seus 12 times estão prontos para trabalhar.

Como Usar os Squads

⚡ Padrão de uso

Todo comando segue o formato: /[squad]:[tipo]:[nome]

exemplos 
# Ativar o Gary Halbert para escrever copy:
/copy-master:agents:gary-halbert

# Pedir ao Brand Squad para gerar nomes:
/brand-squad:tasks:generate-names

# Convocar o Advisory Board para uma decisão:
/advisory-board:agents:ray-dalio

# Análise de performance de tráfego:
/traffic-masters:tasks:analyze-performance

# Criar sistema de design completo:
/design-squad:tasks:create-design-system

# Auditoria de cybersecurity:
/cybersecurity:tasks:assess-security

Resolução de Problemas

⚠️ "npm: command not found"

O Node.js não foi instalado corretamente ou o terminal não foi reiniciado após a instalação. Feche e abra o terminal novamente e tente node --version.

⚠️ Squads não aparecem após instalar

Feche o Claude Code completamente e abra novamente. O Claude Code carrega os comandos apenas na inicialização.

⚠️ "Permission denied" no cp

Verifique se a pasta ~/.claude/commands/ existe. Crie com: mkdir -p ~/.claude/commands e tente novamente.

⚠️ "git: command not found"

O Git não está instalado. Volte ao Passo 0b e complete a instalação do Git antes de continuar.
